Данный проект содержит следующие файлы:
Папка data -  содержит все необходимые классы, для работы с БД и класс для Формы Регистрации
Папка db - содержит файл БД. Если его еще нет - будет создан автоматом
Папка static - хранит все статичные элементы приложения. Стили для украшения страниц, картинки и т.д.
Папка templates - хранит все html шаблоны страниц, какие мы можем запускать в приложении

Для запуска приложения надо первоначально заполнить базу. Для этого.
1. Запустить файл main_users. Он создаст БД и запишет туда несколько пользователей.
2. Запустить файл main_jobs. В уже созданную БД запишет несколько работ, назначенных пользователям.

! НЕ ДОБАВЛЯЙТЕ РАБОТЫ С НЕСУЩЕСТВУЮЩИМИ id  тимлида. Проверяйте в БД, какие id у пользователей у вас есть.
! и не запускайте приложение с пустой базой


Файл main содержит алгоритмы, для АПИ. Благодаря нему можно запустить свой сервер и посмотреть, как запросы отображаются
на html страницах.
